## 1. 取得する情報

本アプリは、以下の情報を取得・処理します。

| 区分 | 具体的な内容 | 取得方法 |
|------|------------|---------|
| アカウント情報 | ユーザー識別子（UID）、メールアドレス、表示名（提供を受けた場合） | Apple または Google アカウントによるサインイン |
| 家計データ | 収支の記録（金額・日付・カテゴリ・品目名・メモ）、家計簿の名称・設定 | 利用者による入力、またはレシート読み取り結果 |

| 広告関連情報 | 広告識別子（IDFA / 広告 ID）、広告配信に必要な情報 | 無料プランでの広告表示時（後述） |
| 端末・利用情報 | アプリの動作に必要な範囲の端末情報 | 自動取得 |

## 2. 利用目的

取得した情報は、以下の目的でのみ利用します。

1. 本アプリの基本機能（家計簿の記録・表示・集計・同期）の提供
2. レシート画像の文字認識（OCR）およびカテゴリ自動分類の処理
3. アカウントの認証・管理、複数端末間でのデータ同期
4. サブスクリプションの提供・状態管理
5. 無料プランにおける広告の配信
6. 不具合の調査・品質改善、お問い合わせ対応
7. 利用規約違反・不正利用への対応

## 3. 外部サービスへの情報送信・第三者への提供

本アプリは、機能提供のため外部の事業者のサービスを利用しており、目的の達成に必要な範囲で情報を外部に送信します。本条は、個人情報保護法および電気通信事業法（外部送信規律）に基づく開示を兼ねます。各事業者における取り扱いは、それぞれのプライバシーポリシーに従います。

### 3-1. 業務委託先への提供（処理の委託）

以下は、本アプリの機能を提供するための処理を委託するものであり、当方は委託先に対し必要かつ適切な監督を行います。

| 送信先（委託先） | 送信される情報 | 利用目的 |
|------------------|--------------|----------|
| Google Cloud Vision API（Google LLC） | レシート画像 | 文字認識（OCR） |
| Anthropic（Claude API / Anthropic, PBC） | レシート画像・抽出テキスト（有料プラン時のみ） | 品目のカテゴリ・税率の自動判定 |
| Firebase Authentication / Cloud Firestore（Google LLC） | アカウント情報・家計データ | 認証・クラウド保存・端末間同期 |

- レシート画像は、上記 OCR・分類処理の目的でのみ外部 API へ送信し、当方のサーバー（Cloud Firestore 等）には保存しません。処理後は当方において保持しません。
- Cloud Vision・Anthropic の各 API は、送信データが各社の生成 AI モデルの学習に利用されない条件の商用利用枠を使用しています。各社における取り扱いの詳細は、それぞれのプライバシーポリシー・API 規約に従います。

### 3-2. 外部送信（広告・効果測定）

無料プランでは、以下の事業者へ情報が送信されます（電気通信事業法の外部送信規律に基づく開示）。

| 送信先 | 送信される情報 | 利用目的 |
|--------|--------------|----------|
| Google Mobile Ads（AdMob / Google LLC） | 広告識別子（IDFA / 広告 ID）、端末・利用に関する情報 | 広告の配信・表示・効果測定 |

- 詳細は AdMob のプライバシーとデータの取り扱い（ https://policies.google.com/technologies/partner-sites ）をご確認ください。
- 有料プラン（Light / Premium）では広告は表示されず、本送信は行われません。

### 3-3. 外国にある第三者への提供（越境移転）

上記の Google および Anthropic は、日本国外（米国を含む）に所在し、または国外のサーバーで情報を処理する場合があります。利用者は、本アプリの利用により、これら**外国にある第三者へ情報が移転されること**に同意するものとします。移転先国の個人情報保護制度や各社の安全管理措置に関する情報は、各社の公表情報および当方への請求により確認できます。

### 3-4. その他

上記のほか、以下の場合を除き、利用者の同意なく個人情報を第三者へ提供しません。

## 4. 広告について（無料プラン）

無料プランでは、Google AdMob による広告を表示します。AdMob は広告配信・効果測定のために広告識別子等を利用する場合があります。

- iOS では、アプリのトラッキング透明性（ATT）に基づき、トラッキングの許可を求める場合があります。許可しない場合でも本アプリの基本機能は利用できます。
- 広告のパーソナライズに関する設定は、端末の設定（iOS:「トラッキング」/ Android:「広告」）から変更できます。
- AdMob のデータ取り扱い: https://policies.google.com/privacy

有料プラン（Light / Premium）では広告は表示されません。

## 5. データの保存場所・保存期間

- 家計データ・アカウント情報は、Google（Firebase / Cloud Firestore）のサーバー上に保存されます。
- 一部の設定情報（プラン状態・カテゴリ設定等）は端末内（ローカルストレージ）に保存されます。
- データは、利用者がアカウントを削除するまで、または法令上必要な期間、保存されます。

---

## 6. データの削除・開示等の請求

- 利用者は、本アプリ内の操作により記録データを削除できます。
- アカウントおよび関連データは、本アプリ内の「設定」→「アカウントを削除」から、利用者自身でいつでも削除できます。削除により、家計データ・アカウント情報は当方の保存先（Cloud Firestore 等）から消去されます。
- アプリを利用できない場合など、上記連絡先へのご連絡による削除も受け付けます。当方は、本人確認のうえ、合理的な期間内に対応します。
- 保有個人データの開示・訂正・利用停止等の請求についても、上記連絡先で受け付けます。
